vue和react的比较

组件化方式不同：

  1. vue通过 .vue文件，来创建相应的组件
    • template 结构
    • script 组件
    • style 样式
  2. react中有组件化的概念，但没有组件模块文件，react中一切都是以JS来表现的

React概览

核心思想：

封装组件 (各个组件维护自己的状态和UI 当状态改变后自动更新组件)

React概念组成：

  • 组件
  • JSX (JS支持HTML的嵌入)
  • Virtual DOM
  • Data Flow (单向数据绑定)
